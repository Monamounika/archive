rem.py - removes packages from pk2.txt ( prints to stdout ) from data in remove.txt

yi.sh - install packages provided by yaourt-packages.txt

final.txt - cleaned up pacman packages list generated by rem.py

remove.txt - packages that belong to the AUR that should not be in final.txt ( known to not be in the Official Arch repos )

yaourt-install.sh - installs yaourt and deps on arch linux systems

aurpack.install - bash install script generated by aurgen.sh > aurpack.install

install-linux49.sh - bash install script to install linux49 and deps

if having trouble at this directory level, try the packs2 directory level
